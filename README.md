# 配置 node 命令行有道翻译



![](https://github.com/tomatobybike/Tomato-Drawing/blob/main/youdaotom.jpg?raw=true)

## Install

```
$ yarn global add youdaotom

```

## Or install

```
yarn add youdaotom
```

## Usage

```sh
youdaotom query hello
```

## short cmd

```sh
youdaotom  hello
```

## set key and fromkey

```sh
youdaotom set -key 'xxx' -keyfrom 'xxx'
```
